## Beli-Hamburger (BH)

Website ini adalah Landing Page yang sangat sederhana, yang dibuat dengan menggunakan:
- HTML
- CSS
- Javascript

Di desain menggunakan tools atau aplikasi Figma. Untuk melihat desain, silahkan kunjungi [Design Beli-Hamburger](https://www.figma.com/file/DPwyNstEOBMTaeVtDeGVLa/Beli-Hamburger?node-id=0%3A1&fbclid=IwAR33iIBg3BsGvN_mWsgncQt6u39y6MAo6WodujDopTUP5bgnxQmH59DYZIg) 

Untuk melihat demo nya silahkan kunjungi [Demo](https://beli-hamburger.netlify.app/)

Saya membuat website ini hanya sebatas untuk belajar lebih lanjut tentang CSS dan Javascript, sekaligus belajar agar dapat terbiasa dengan menggunakan text editor Vim.

Apabila terdapat kekurangan pada tampilan saat dibuka di beberapa perangkat kalian, silahkan berikan masukkan nya di tab **Issues**
